Contact: <project email or URL>

## Notes
- Remove any "All rights reserved" language when using Creative Commons; it conflicts with the license grant.
- Keep the license choice consistent across all documents, or explicitly note exceptions (e.g., figures under separate terms).
- Add cross-links: GitHub repo URL, OSF and Zenodo DOIs.
