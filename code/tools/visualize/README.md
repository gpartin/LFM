Visualization tools
===================

This folder contains standalone visualization scripts for inspecting and
presenting results (GIF/MP4/PNGs) generated by the Tier harnesses.

Notes
- These scripts expect to be run from the repository root so relative
  paths like `results/...` resolve correctly.
- They don’t depend on test discovery; they’re plain Python scripts you
  can call directly.

Examples
- Optional:
  - python tools/visualize/visualize_grav12.py
  - python tools/visualize/visualize_grav12_phase_group.py
  - python tools/visualize/visualize_grav15_3d.py --input results/Gravity/GRAV-15/diagnostics/field_snapshots_3d_GRAV-15.h5
  - python tools/visualize/visualize_grav16_camera.py --input results/Gravity/GRAV-16/diagnostics/field_snapshots_3d_GRAV-16.h5
  - python tools/visualize/visualize_grav16_doubleslit.py --input results/Gravity/GRAV-16/diagnostics/field_snapshots_3d_GRAV-16.h5
  - python tools/visualize/visualize_hamiltonian.py --tests ENER-05 ENER-06 ENER-07
