## TROUBLESHOOTING

### Problem: Upload fails or times out
**Solution**: 
- Check file size (should be < 5 MB for markdown files)
- Try uploading one file at a time
- Use wired internet connection if available
- Clear browser cache and try again

### Problem: DOI not assigned
**Solution**:
- OSF DOIs are usually automatic for public projects
- Check project settings ‚Üí DOI section
- May need to request DOI if project was private initially
- Contact OSF support: support@osf.io

### Problem: Files won't display/preview
**Solution**:
- Markdown files may not preview in-browser (this is normal)
- Downloadable is more important than previewable
- Consider converting to PDF if preview is critical
- OSF supports .md, .pdf, .txt, .doc, .docx

### Problem: Project not appearing in search
**Solution**:
- Google indexing takes 24-72 hours
- Google Scholar takes 48-96 hours
- Make sure project is PUBLIC (not private or embargoed)
- Add more descriptive tags and keywords
- Share direct link on social media to speed up discovery

### Problem: Can't edit project description
**Solution**:
- Make sure you're logged in as project owner
- Check that project isn't in read-only mode
- Try different browser (Chrome/Firefox/Edge)
- Contact OSF support if persistent
